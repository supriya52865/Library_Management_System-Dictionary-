# Library Management System

## Project Description

The Library Management System is a menu-driven Python application developed using a modular package architecture. The project manages library operations such as adding books, viewing available books, issuing books, and returning books.

Book records are stored using Python dictionaries, where each book is assigned a unique Book ID along with its details. The program follows a structured design by separating functionality into different modules, making the code easy to maintain and understand.

### Features

* Add new books to the library
* Display all available books
* Issue books to users
* Return issued books
* Menu-driven interface using an infinite loop
* Dictionary-based book record management
* Modular package architecture
* Simple and user-friendly console interface

### Technologies Used

* Python
* Dictionaries
* Functions
* Modules and Packages
* Conditional Statements
* Loops

### Project Structure

* `main.py` – Main menu and program execution
* `add_books.py` – Add new books
* `show_books.py` – Display available books
* `issue_book.py` – Issue books
* `return_book.py` – Return issued books
* `utils.py` – Data storage using dictionaries

### Learning Outcomes

This project demonstrates the use of Python dictionaries for data management, modular programming, package architecture, and menu-driven application development. It helps in understanding how real-world library operations can be implemented using fundamental programming concepts.
